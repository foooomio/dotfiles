# dotfiles
My dotfiles

## Installation
```bash
git clone https://github.com/foooomio/dotfiles.git && cd dotfiles && ./setup.sh install
```
Original files will be backed up in `~/.backup`.

## Optional Installation

### Vim plugins
```bash
./setup.sh dein
```

### MacVim
```bash
./setup.sh homebrew
```

### Homebrew
```bash
./setup.sh homebrew
```

## Uninstallation
```bash
./setup.sh uninstall
```
